如何搭建Hexo+Github博客

Veröffentlicht am 2018-09-28 | Edited on 2018-09-29

前言

一个前端小白,用来记录前端学习历程,在网上查了诸多资料,搭建起了独立的博客,独立的才是自己的.
现在写下此文,分享给大家,过程很繁琐,耐住性子.

环境搭建

Node.js 的安装和准备

  • Node.js 去官方网站 根据操作系统去下载即可
  • 然后一路next 傻瓜式安装

  • 打开cmd黑窗口 验证是否安装成功

    • 验证node

    • 验证npm

  • 返回版本号即为安装成功,npm是和node一块安装上的

git 的安装和配置

  • git 去官方网站 根据操作系统去下载即可
  • 同上 傻瓜式安装
  • 在桌面空白处右击 出现 Git Bash Here 和 Git GUI Here 即可

github 账户的配置

  • github官网 注册账号
  • 登录后点击右上角加号 选择 New repository 创建代码库
  • 在Repository name下填写yourname.github.io, yourname必须和你的账户名保持一致 Description (optional)下填写一些简单的描述（不写也没有关系）
